Project Name: CreditWithoutBarriers 

Type:
Full-Stack FinTech Web Application
(PHP + MySQL + Bootstrap + OTP + PDF Generator)

How to Install:
1. Copy folder "a" into C:\xampp\htdocs\
2. Open XAMPP and start Apache + MySQL
3. Open phpMyAdmin → Create database: database1
4. Import the file database1.sql
5. Open browser: http://localhost/a/

Roles:
Admin Login: adminlogin.php
Lender Login: lenderlogin.php (role: lender)
Borrower Login: customerlogin.php (role: borrower)
